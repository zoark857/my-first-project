# Multi Number Calculator

My first Python calculator project.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulo (%)
- Multiple number input
- Decimal support
- Division by zero handling

## How to Run

1. Install Python.
2. Download calculator.py.
3. Run:

python calculator.py

## Example

Enter numbers with spaces:
10 20 30

Enter operation:
+

Result:
60.0
